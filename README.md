**Это zapret с интерфейсом**. У вас что-то сломалось?
Вы устали заходить в нужную папку, выбирать нужный bat файл из-за очередной блокировки? 
Теперь это все в одной программе.

- No config - это значит вы скачиваете обычный .exe файл и перемещаете его в папку с zapret. (**в разработке**)

### Установка и первый запуск
1. Перейдите в раздел [Releases](https://github.com/Savvy08/Zapret-Custom/releases/)
2. Скачайте последнюю версию `ZapretCustom_Setup.exe`
3. Запустите установщик и следуйте инструкциям
4. После установки программа появится на рабочем столе (если выбрали эту опцию)
5.  **Запустите программу** с правами администратора
6. **Настройте конфигурацию:**
   - Нажмите "НАСТРОИТЬ КОНФИГ"
   - Выберите подходящий `.bat` файл из списка
   - Нажмите "Выбрать"
7. **Запустите Zapret:**
   - Нажмите "ЗАПУСТИТЬ ZAPRET"
   - Дождитесь появления статуса "ВКЛЮЧЁН"
8. **Проверьте соединение:**
   - Нажмите "ТЕСТ СОЕДИНЕНИЯ"
   - Убедитесь, что Discord и YouTube доступны (есть БАГ, все время показывает discrod якобы не работает, смотрите по доменну)
  
### Основные файлы (в папке `lists`)
- `ipset-all.txt` - Все IP адреса
- `ipset-exclude.txt` - Исключённые IP
- `list-exclude.txt` - Исключённые домены
- `list-general.txt` - Основной список доменов
- `list-google.txt` - Домены Google

### DLC файлы (в папке `lists/DLC`)
- `ipset-cloudflare.txt` - IP адреса Cloudflare
- `ipset-ea.txt` - IP адреса EA
- `list-general (2).txt` - Дополнительный список доменов

### Как редактировать списки:

1. Откройте **Настройки** → **Настроить списки**
2. Выберите нужный файл из списка
3. Нажмите **"Открыть в блокноте"**
4. Внесите изменения и сохраните файл
5. Перезапустите Zapret для применения изменений

### Важные замечания!

#### Права администратора
Программа **ОБЯЗАТЕЛЬНО** должна запускаться с правами администратора для:
- Изменения сетевых настроек
- Запуска/остановки службы Zapret

#### Обновление Zapret
Для обновления самого Zapret (не GUI):
1. Откройте **Настройки**
2. Нажмите **"Настроить service.bat"**
3. Запустите скрипт обновления

### Решение проблем

#### Программа не запускается
- Убедитесь, что запускаете с правами администратора
- Проверьте наличие файла `icon.ico` в папке программы
- Убедитесь, что все `.bat` файлы находятся в корневой папке

#### Zapret не работает после запуска
- Проверьте, что выбрана корректная конфигурация
- Запустите тест соединения для диагностики
- Попробуйте другую конфигурацию из списка
- Проверьте логи в консоли Zapret

#### Discord/YouTube не открываются
- Подождите 10-15 секунд после запуска Zapret
- Попробуйте очистить кэш браузера
- Перезапустите браузер
- Попробуйте другую конфигурацию

#### "Не найден файл python314.dll"
Эта ошибка говорит о повреждённой установке. Решение:
- Удалите программу полностью
- Скачайте установщик заново
- Переустановите программу
